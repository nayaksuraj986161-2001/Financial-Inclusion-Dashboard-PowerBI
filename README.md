#  Financial Inclusion Dashboard using Power BI

##  Project Overview

This project focuses on designing an interactive Power BI dashboard using the Financial Inclusion in Africa dataset. The dashboard provides insights into bank account ownership, demographic trends, education levels, and geographical factors influencing financial inclusion across East African countries.

The objective of this project is to help business stakeholders and policymakers understand patterns of financial inclusion and support data-driven decision-making.

---

##  Objective

- Design an interactive dashboard for business stakeholders.
- Analyze bank account ownership across different demographic groups.
- Identify factors affecting financial inclusion.
- Present insights through visual analytics and KPI indicators.
- Enable interactive filtering and exploration of data.

---

##  Dataset Information

**Dataset Name:** Financial Inclusion in Africa

**Source:** Kaggle

**Total Records:** 23,524 Respondents

### Countries Covered
- Kenya
- Rwanda
- Tanzania
- Uganda

### Features Included
- Country
- Year
- Gender of Respondent
- Respondent Age
- Type of Location
- Education Level
- Marital Status
- Job Type
- Cell Phone Access
- Bank Account Ownership

##  Tools Used

- Power BI Desktop

##  Key Performance Indicators (KPIs)

The dashboard includes the following KPI cards:

| KPI | Description |
|------|-------------|
| Total Respondents | Total survey participants |
| Bank Account Holders | Number of respondents with bank accounts |
| Financial Inclusion Rate | Percentage of account holders |
| Average Age | Average age of respondents |

---

##  Dashboard Components

### 1. Bank Account Ownership Analysis
- Donut Chart
- Displays respondents with and without bank accounts.

### 2. Country-wise Analysis
- Clustered Bar Chart
- Compares financial inclusion across countries.

### 3. Urban vs Rural Analysis
- Column Chart
- Compares banking access based on location type.

### 4. Gender-wise Analysis
- Stacked Column Chart
- Shows account ownership by gender.

### 5. Education Level Analysis
- Bar Chart
- Examines the relationship between education and financial inclusion.

### 6. Age Group Distribution
- Column Chart
- Shows respondent distribution across age groups.

---

##  Interactive Features

The dashboard includes slicers for:

- Country
- Year
- Gender
- Location Type
- Education Level

These filters allow users to interactively explore the dataset and generate customized insights.

---

##  Key Findings

### Financial Inclusion Status
- Total Respondents: 23,524
- Bank Account Holders: 3,309
- Financial Inclusion Rate: 14.07%
- Average Age: 38.80 Years

### Country Insights
- Kenya has the highest level of bank account ownership.
- Rwanda, Tanzania, and Uganda show lower financial inclusion levels.

### Demographic Insights
- Urban respondents are more likely to own bank accounts.
- Higher education levels correlate with increased financial inclusion.
- Financial inclusion varies significantly across age groups.

## Business Recommendations

- Expand banking services in rural areas.
- Promote financial literacy and awareness programs.
- Encourage digital banking adoption.
